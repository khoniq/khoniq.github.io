Sentimental Value

dir. Joachim Trier
2026-02-11 

“We can’t really talk.”

即便是劇中劇這麼無聊的 trick，JT 的節奏感還是這麼令人安心。轉眼間低彩度黑酷 Reprise 要20年了，沈澱至今浮現伯格曼味（好的那種），不確定是為什麼。 但可以確定的是，Joachim Trier、Eskil Vogt 兩個人太會寫。從高度自覺越嶺到企圖和解，天曉得心裡吃了多少苦頭。


雖說 The Worst Person in the World 廣受好評，私心喜歡這部更多。 也許是年紀到了，作者的視角感覺距離兩代人並非不成比例地靠近或遠離，而是處在中間的平衡位置。 除了本來就常做的自我挖苦/懷疑、後設，JT 開始會灑上一些 inside jokes，相較以往作品用放大鏡看存在焦慮，慢慢開始展現鬆弛感。


千古不朽的家庭主題，很高明地在許多地方好像什麼都沒說，卻也都說完了。默認/默契是家庭之間最常見的奇異現象，極度渴望溝通卻難以言明的痛苦和掙扎，不耽溺也不放過。


網評說過去誤以為 Barry Jenkins 是最善解人意 (compassionate) 的導演，至今才發現 JT 可能更適合這個封號。 雖然我不確定這是哪來的斷言，但是對於本片深藏的 compassion 有此感受，著實相當容易理解。（4/5）
